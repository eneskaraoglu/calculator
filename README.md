# Calculator Mobile App

A Flutter-based calculator application built for mobile devices. This project
focuses on a clean user interface, fast interaction, and the core behaviors
users expect from a basic calculator app.

## Project Overview

This app is a simple calculator designed with Flutter and Material 3. It runs
on Android and can also be launched on other Flutter-supported platforms such as
web and desktop during development.

The current version includes:

- Basic arithmetic operations: `+`, `-`, `*`, `/`
- Decimal number input
- Percentage conversion
- Sign toggle with `+/-`
- Delete last character with `DEL`
- Full reset with `C`
- Live result preview while typing
- Dark themed modern calculator interface

## Main Goal

The goal of this project is to create a beginner-friendly but practical mobile
calculator app that can be improved over time. It is a good foundation for:

- Learning Flutter UI development
- Understanding state management with `StatefulWidget`
- Building calculator logic manually
- Expanding a mobile app with more advanced features later

## How the App Works

The app is built around a single calculator screen with two main parts:

1. Display area
   Shows the current expression and the current calculated result.

2. Button grid
   Provides number keys, operator keys, and utility actions like clear, delete,
   percent, sign change, and equals.

### State Management

The calculator uses a `StatefulWidget` to update the screen whenever the user
presses a button. Two state values are tracked:

- `_expression`
  The text the user is currently typing

- `_result`
  The calculated output based on the expression

### Input Handling

When a user taps a button:

- Numbers are appended to the expression
- Operators are appended or replace the previous operator
- Decimal input is limited so a number cannot contain multiple dots
- `DEL` removes the last character
- `C` resets everything
- `%` converts the current number into a percentage
- `+/-` changes the sign of the current number
- `=` finalizes the calculation result

### Calculation Logic

The calculator evaluates arithmetic expressions manually in Dart without using
an external math parser package.

It works by:

- Tokenizing the expression into numbers and operators
- Respecting operator precedence
- Handling unary negative values
- Returning `null` for invalid expressions
- Preventing division by zero

The final numeric output is formatted so whole numbers appear cleanly without
extra decimal places.

## User Interface Design

The UI uses:

- A dark green color palette
- Rounded cards and buttons
- A gradient result panel
- Large text for readability
- A responsive button grid suitable for mobile screens

This gives the calculator a modern mobile-app look while staying simple.

## How to Run the Project

Make sure Flutter is installed and available in your `PATH`.

### Check Flutter

```powershell
flutter --version
flutter doctor -v
```

### Get Dependencies

From the project root:

```powershell
flutter pub get
```

### Run on Android Emulator

Start your Android emulator first, then run:

```powershell
flutter run
```

Or explicitly:

```powershell
flutter run -d <device-id>
```

### Run on Web

```powershell
flutter run -d chrome
```
